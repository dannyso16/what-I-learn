## CHAPTER3

### ビューとビューグループ

ビューグループ：配置を決める（レイアウト部品ともいう）

| タグ               | 内容                   |
| ------------------ | ---------------------- |
| <LinearLayout>     | 縦or横に並べて配置     |
| <TableLayout>      | 表形式で配置           |
| <GridLayout>       | グリッド形式           |
| <FrameLayout>      | 画面部品を重ねて配置   |
| <ConstraintLayout> | 画面部品を相対的に配置 |

ビュー：画面部品そのもの（ウィジェット）

| タグ          | 内容い                                       |
| ------------- | -------------------------------------------- |
| <TextView>    | 文字列の表示                                 |
| <EditText>    | テキストボックス（一行や数字のみなども可能） |
| <Button>      | ボタン                                       |
| <RadioButton> | ラジオボタン                                 |
| <CheckBox>    | チェックボックス                             |
| <Spinner>     | ドロップダウンリスト                         |
| <ListView>    | リスト表示                                   |
| <SeekView>    | スライダー                                   |
| <RatingBar>   | ★でレートを評価                              |
| <Switch>      | On/Offが表現できるやつ                       |
